# useI18n для интернационализации Vue приложений

## Обзор
Функция `useI18n` предоставляет простой и эффективный способ добавления в проект мультиязычной поддержки.

Состав функции:
- файл `useI18n.ts` содержит непосредственно код функции
- папка `locales` хранит файлы формата `<lang>.ts` с объектами, содержащими строковые значения переводов.

У функции есть два метода:
- метод `initI18n` инициаплизирует перевод с текущим кодом языка
- метод `t(someObject.someMessage)` возвращает значение, в данном случае булет искать в объекте `someObject` поле `someMessage`

## Как работает

Функция импортируется в `RootComponent`, которыйпринимает в пропс `locale`  с бэка текущий язык страницы (в виде двузначного кода, например `ru`).
При монитировании `RootComponent` в `onMounted` вызывается метод функции `initI18n`, он принимает этот код и загружает файл перевода из папки `/locales/` (в нашем случае перевод подтянется из `locales/ru.ts`).

::: info ВАЖНО
Чтобы магия сработала, нужно чтобы с бэка пришёл двузначный код, который совпадает с именем файла перевода.
:::

::: tip ЗАМЕТОЧКА
По умолчанию в шаблоне нет мультиязычности, поэтому в `RootComponent` закомментированы все ненужные импорты и инициализации. Чтобы всё заработало, нужно раскомментировать импорты и вызовы.
:::

## Пример использования

В примере ниже
- компонент импортирует функцию `useI18n`
- происходит её инициализация (если кода ещё нет - то с кодом `ru`)
- из файла `ru.ts` подтягиваются переводы для `test.hello` и `test.jobTitle`
- по клику на кнопку "Поменять язык" выполняется функция `initLanguage`, которая меняет значение переменной и заново инициализирует initI18n

При работе с бэкендом смена языка может сопровождаться перезагрузкой страницы, тогда повторно инициализировать ничего не нужно.
В данном примере ничего не перезагружается, поэтому в `initLanguage` мы каждый раз производим инициализацию, подтягивая свежие данные,.

### Код

```vue
<script setup>
import { useI18n } from '@/app/shared/composables/useI18n/useI18n.ts';
import { ref } from 'vue';

const { t, initI18n } = useI18n();
const lang = ref('');

function initLanguage(lang: string) {
	lang.value === 'ru' ? (lang.value = 'en') : (lang.value = 'ru');
	initI18n(lang.value);
}

initI18n(lang.value);
</script>

<template>
	<div>
		<button @click="initLanguage">Поменять язык</button>
		<h4>{{ t('test.hello') }}{{ t('test.jobTitle') }}</h4>
	</div>
</template>
```
