# Tooltip

Pug-миксин для отображения всплывающих подсказок с поддержкой разных позиций относительно триггера.

Расположение: `src/templates/shared/tooltip.pug`

Поддерживает
- кастомный триггер для появления
- кастомный контент для отображения
- позиционирование

## API

### Аргументы

| Название   | Тип      | По умолчанию | Описание              |
| ---------- | -------- | ------------ | --------------------- |
| `id`       | `string` | `'tooltip'`  | Уникальный id тултипа |
| `position` | `string` | `'bottom'`   | Позиция тултипа       |

### Поддерживаемые позиции
| Значение       | Описание         |
| -------------- | ---------------- |
| `top`          | Сверху по центру |
| `top-right`    | Сверху справа    |
| `top-left`     | Сверху слева     |
| `bottom`       | Снизу по центру  |
| `bottom-right` | Снизу справа     |
| `bottom-left`  | Снизу слева      |
| `right`        | Справа           |
| `left`         | Слева            |

## Слоты
- `trigger` - кастомный элемент-триггер для отображения тултипа (если слот не передан — отображается иконка info)
- `content` - содержимое тултипа

## Использование

::: warning Важно
- Для каждого тултипа на странице нужно использовать уникальный id
- Нелья размещать интерактивные элементы внутри тега button
- Для сложного\большого контента нужно задавать min-width
:::

### Базовый тултип

```pug
+tooltip({ id: 'tooltip-basic' })
	+slot('content')
		.tooltip-card
			span Полезная подсказка
```

### Тултип с кастомным триггером

```pug
+tooltip({ id: 'tooltip-custom' })
	+slot('trigger')
		div.btn Подробнее
	+slot('content')
		div.tooltip-card
			| Текст подсказки
```

## Особенности
- Компонент использует нативный popover
- Если тултип не помещается в выбранной позиции — браузер автоматически попытается переставить его
- Контейнер тултипа не стилизован: нужно либо стилизовать его, либо пробрасывать уже стилизованный контент
