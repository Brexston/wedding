# Иконки

Для интерфейсных иконок (в кнопках, ссылках и т.п.) рекомендуется использовать **SVG-спрайты**. Это оптимальное решение, когда иконки не требуется менять через админку — например, на карточках, кнопках, в меню.  
Однако **не стоит использовать спрайты** в тех случаях, где иконки должны быть редактируемыми из CMS — например, блоки социальных сетей.

## Цвет иконок

Чтобы иконки можно было легко перекрашивать через CSS, необходимо соблюдать следующие правила при работе с SVG:

- Удаляйте атрибуты `fill`, `stroke` и подобные из самого SVG-файла.
- Внутри SVG-элементов прописывайте `stroke="currentColor"` — это позволит управлять цветом иконок с помощью `color` в CSS.
- По возможности избегайте `fill` — используйте `stroke`, если стиль иконки это позволяет.

Такой подход обеспечит максимальную гибкость в стилизации через классы.

## Структура проекта

SVG-иконки, используемые в спрайте, следует размещать по пути:

```
/src/assets/icons/
```
