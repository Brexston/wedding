# Шрифты

## Подключение через `unplugin-fonts` (базовый способ)
В шаблоне используется плагин [`unplugin-fonts/vite`](https://www.npmjs.com/package/unplugin-fonts) для автоматического подключения локальных шрифтов. Он работает совместно с функцией `getFonts`, которая сканирует папку со шрифтами и передаёт данные плагину.

Кратко: все, что нужно это создать папку семейства шрифта и переместить шрифты в эту папку `/src/public/fonts/название_семейства_шрифта`

### 📂 Структура папки

Шрифты располагаются в директории:

```
/src/public/fonts/имя_семейства_шрифта
```

Каждое семейство — это отдельная папка, внутри которой находятся файлы шрифтов.

:::danger ВАЖНО!!!
- Файлы шрифтов **обязаны** соответствовать следующему формату:  
  `ИмяШрифта-Начертание.формат`

  **Пример:**  
  `Montserrat-Regular.woff2`

- Название папки должно точно совпадать с именем шрифта:  
  `Montserrat` → `Montserrat-Regular.woff2`
:::

## Как подключают шрифты сеньоры
:::tip Чем этот способ лучше?
- Шрифты устанавливаются через `npm`, не попадают в git-репозиторий.
- Есть версия каждого пакета — удобно для контроля изменений.
- Подключение выполняется нативно, без сторонних плагинов.
- Поддержка вариативных шрифтов (100–900) и отдельных начертаний.
:::

Более надёжный и гибкий способ подключения — использование [Fontsource](https://fontsource.org/). Это набор npm-пакетов со всеми открытыми шрифтами в форматах `woff` и `woff2`, разбитыми по начертаниям и языковым формам (латиница, кириллица и др.).

### ⚙️ Инструкция по установке (на примере Montserrat)

1. Установите пакет шрифта:
   ```sh
   npm install @fontsource/montserrat
   ```

2. В `package.json` **удалите `^`** перед номером версии, чтобы зафиксировать конкретную версию пакета:
   ```json
   "@fontsource/montserrat": "5.0.12" // вместо "^5.0.12"
   ```

3. Подключите шрифт в `main.ts`:
   ```ts
   import '@fontsource/montserrat';
   ```

4. Отключите старое подключение через `unfonts.css` в `main.ts`:
   ```ts
   // import 'unfonts.css';
   ```

5. Отключите плагин `unfonts` в `vite.config.ts`:
   ```ts
   // Unfonts({
   //   custom: {
   //     display: 'swap',
   //     families: FONTS,
   //   },
   // }),
   ```

:::danger ВАЖНО!!!
Папку `/src/public/fonts/` нельзя удалять, она должна быть пустой
:::

Теперь шрифт подключён напрямую из npm, в надёжной и расширяемой форме — как это делают в продвинутых проектах.
