# Что это?

Frontend-шаблон — это готовая к использованию стартовая заготовка для быстрой и удобной разработки веб-приложений. Он
помогает быстро запустить проект с уже настроенной сборкой. Всё, что нужно для вёрстки и написания логики уже готово.

## Что используется внутри

**Для вёрстки**:
- [Pug](https://pugjs.org/api/getting-started.html) — чтобы писать HTML проще и короче.
- [Sass](https://sass-lang.com/) — чтобы стили были гибкими и структурированными.
- [PostCSS](https://postcss.org/) с плагинами для конвертации `px` в `rem` и другой полезной магии.

**Для фронтенда:**
- [Vue 3](https://vuejs.org/) — основной фреймворк, на котором всё строится.
- [TypeScript](https://www.typescriptlang.org/) — добавляет типизацию и делает код надёжнее.
- [Pinia](https://pinia.vuejs.org/) — современное хранилище данных.
- [Axios](https://axios-http.com/ru/docs/intro) — чтобы отправлять запросы на бэкенд.
- [VueUse](https://vueuse.org/) — полезные готовые хуки для типовых задач.
- [Vuelidate](https://vuelidate-next.netlify.app/) — для удобной валидации форм.
- [Vue Router](https://router.vuejs.org/) — для переходов между страницами (для SPA).
- [Floating Vue](https://floating-vue.starpad.dev/), [Vue Toastification](https://vue-toastification.maronato.dev/),
[Vue Datepicker](https://vue3datepicker.com/), [Swiper](https://swiperjs.com/) и др. — компоненты для UI и UX.

**Вспомогательные инструменты:**
- [Vite](https://vite.dev/) — быстрый и удобный сборщик проекта.
- [ESLint](https://eslint.org/) и [Stylelint](https://stylelint.io/) — следят за качеством кода.

В общем, это универсальный шаблон, который подойдёт как для простых лендингов, так и для больших SPA-приложений. Не
нужно тратить время на настройку — можно сразу переходить к разработке.
