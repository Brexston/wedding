# 📁 src

`src/` — это рабочая область проекта, в которой хранится весь исходный код.

Здесь находятся точки входа приложения, модульная архитектура, общие компоненты и утилиты, а также стили, шаблоны и
страницы.

Папка объединяет как frontend-логику (Vue, TypeScript), так и вёрстку (Sass, Pug) с ресурсами (иконки,
изображения, шрифты), обеспечивая полный цикл сборки и рендеринга проекта.
