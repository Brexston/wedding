# 📄 .npmrc

Файл `.npmrc` задаёт конфигурацию менеджера пакетов **npm** для проекта.
В нём можно описывать правила установки зависимостей, настройки реестра, прокси и другие параметры.

## В шаблоне

```text
engine-strict = true
```

Эта настройка включает строгую проверку движка (`engine-strict`).

- При установке зависимостей npm проверяет поле `engines` в `package.json`.
- Если версия Node.js или npm не соответствует указанным требованиям — установка прерывается с ошибкой.

## Зачем нужен

- Обеспечивает, что все разработчики и CI используют совместимые версии Node.js.
- Предотвращает баги, возникающие из-за несовместимости окружений.
- Делает проект более предсказуемым и стабильным.
