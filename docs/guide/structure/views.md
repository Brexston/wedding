# 📁 views

Каталог с конечными Pug-страницами проекта.  
В отличие от `/templates`, где хранятся переиспользуемые фрагменты, здесь находятся собранные страницы, которые включают в себя нужные шаблоны, секции и компоненты. 

Каждый файл в корне каталога — это готовая страница (например, `main.pug`, `cart.pug`, `404.pug`).
