# 📁 app

Основной каталог клиентского приложения.  
Содержит точку входа, конфигурацию Vue, бизнес-модули и все общие утилиты, компоненты и скрипты.

## Структура

### 📁 `modules/`

Бизнес-модули, сгруппированные по доменам (cart, forms, media, user и т.д.).  
Каждый модуль может включать в себя:

- `api/` — HTTP-запросы и их типы.
- `store/` — Pinia-хранилища для состояния модуля.
- `components/` — Vue-компоненты, относящиеся только к этому модулю.
- `*.types.ts` — локальные типы.
- Дополнительно при необходимости: собственные `composables`, `utils` и пр.

Пример: `cart/` — включает API, стор, типы и все компоненты корзины.

- **`RootComponent.vue`** — корневой компонент, который оборачивает всё приложение.  
  Здесь инициализируются глобальные скрипты, передаются пропсы от бэкенда (CSRF-токен, локаль, конфиг капчи, ссылки на
  документы).

### 📁 `shared/`

Переиспользуемые части, которые не привязаны к конкретному модулю:

- **`api/`** — модуль API-клиента для HTTP-запросов (Axios), базовые типы и интерсепторы.
- **`captcha/`** — модуль для работы с провайдерами капчи (reCAPTCHA, Яндекс.Капча).
- **`components/`** — UI-библиотека и общие Vue-компоненты (инпуты, модальные окна, слайдеры, иконки и т.д.).
- **`composables/`** — универсальные хуки (`useCopyToClipboard`, `useMedia`, `useTimer`, локализация `useI18n`).
- **`consts/`** — константы и настройки (маски, viewport, параметры уведомлений).
- **`declarations/`** — TS-декларации для глобальных сущностей (Vue, роутер, Axios).
- **`directives/`** — кастомные Vue-директивы (анимации, фиксированный хедер, скролл и т.д.).
- **`router/`** — настройка Vue Router для SPA: маршруты, middleware и их пайплайн.
- **`scripts/`** — вспомогательные скрипты, подключаемые в `RootComponent` (скролл к хэшу и т.д.).
- **`stores/`** — глобальные Pinia-хранилища (например, для модалок).
- **`types/`** — общие типы, доступные во всём приложении.
- **`utils/`** — утилиты и хелперы (работа с файлами, датами, правилами валидации, Я.Метрика и пр.).  

### Файлы верхнего уровня

- **`app.components.global.ts`** — регистрация глобальных Vue-компонентов, доступных во всём приложении.
- **`app.ts`** — создание и конфигурация Vue-приложения (Pinia, плагины, глобальные директивы).
- **`main.ts`** — точка входа, маунтит Vue-приложение и подключает стили.
