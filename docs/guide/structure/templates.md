# 📁 templates

Каталог с Pug-шаблонами проекта, организованными по уровням абстракции.

## Структура

### 📁 `/base`

Базовые части страницы и инфраструктура шаблонов:
- `_head.pug` — `<head>` со всеми подключениями,
- `_scripts.pug` — блок скриптов,
- `_mixins.pug` — глобальные миксины,
- `_modals.pug` — модальные окна, доступные во всём проекте.

### 📁 `/shared`

Переиспользуемые секции, блоки и элементы (хедер, футер, пагинация, хлебные крошки, ошибки, аккордеоны, UI-утилиты и т.п.).

### 📁 `/emails`

Фрагменты для email-шаблонов: баннеры, карточки, таблицы, блоки с текстом, социальные кнопки и пр.

### 📁 `/<page>`

Шаблоны для конкретных страниц или крупных секций, например:
- `main/` — главная страница,
- `content/` — типовая контентная страница.

При добавлении новой страницы создаётся собственная папка.
