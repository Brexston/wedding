# 📁 public


Папка для **статичных публичных ресурсов**, которые не проходят обработку сборщиком и копируются в итоговую сборку «как есть».  
Используется для файлов, доступных напрямую по URL (например, `/img/logo.svg` или `/favicon.ico`).

## Структура

### 📁 `favicons/`

Иконки и манифесты для браузеров и PWA: `favicon.ico`, `favicon.svg`, `apple-touch-icon.png`, webmanifest и изображения для установки приложения.

### 📁 `fonts/`

Шрифты в формате `.woff2`, сгруппированные по семействам (например, `Helvetica`, `Montserrat`), для подключения через CSS.

### 📁 `img/`

- `content/` — баннеры, фотографии, декоративные картинки, соц. иконки для статей/страниц.
- `email/` — картинки, используемые в шаблонах писем (лого, кнопки магазинов, соцсети, плейсхолдеры продуктов).
- `socials/` — иконки соцсетей для использования на сайте.
- отдельные файлы (`logo.svg`, `opengraph-image.jpg`, `404.svg`, `502.svg`) — ключевые изображения проекта (логотип, OG-превью, страницы ошибок).

---

Таким образом, `/src/public` — это место для ресурсов, которые должны быть доступны в исходном виде без оптимизаций и преобразований, в отличие от `/src/assets`, где хранятся иконки и декоративные элементы, встроенные в сборку.
