# 📁 assets

Папка для хранения **статичных ресурсов**, которые используются в интерфейсе.  
Основное назначение — хранение SVG-иконок, подключаемых в компонентах или шаблонах.

* В текущей структуре содержит набор базовых иконок (стрелки, чекбоксы, системные статусы, элементы управления и т. д.).
* Файлы из `assets` обрабатываются сборщиком (Vite) и могут быть оптимизированы или встроены в бандл.
* Папка может опционально использоваться не только для иконок, но и для других **неизменяемых декоративных изображений
  или ресурсов**, которые должны оставаться стабильными (например, графические элементы интерфейса, svg-маски,
  иллюстрации).

Таким образом, `src/assets` — это центральное хранилище для всех небольших статичных графических ресурсов проекта.
