# 📄 .nvmrc

Файл `.nvmrc` используется [Node Version Manager (nvm)](https://github.com/nvm-sh/nvm) и аналогичными инструментами для
указания версии Node.js, с которой должен работать проект.

## В шаблоне

```text
lts/jod
```

Это означает, что проект должен запускаться на **последней актуальной LTS-версии Node.js с кодовым именем “Jod”**.  
При выполнении команд `nvm use` или `nvm install` автоматически будет выбрана/установлена нужная версия.

## Зачем нужен

- Гарантирует, что все разработчики и CI используют одну и ту же версию Node.js.
- Избегает ошибок, связанных с несовместимостью версий.
- Упрощает onboarding новых участников команды (достаточно выполнить nvm use).
