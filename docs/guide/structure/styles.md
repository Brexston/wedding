# 📁 styles

Каталог со всеми стилями проекта, организованными по уровням абстракции.

## Структура

### 📁 `/base`

Глобальные стили и настройки: переменные, миксины, сброс, типографика, анимации и другие элементы,
которые формируют основу дизайн-системы.

### 📁 `/shared`

Стили для переиспользуемых компонентов и общих секций (UI-элементы, шапка, подвал, модальные окна,
блоки ошибок и т.п.).

### 📁 `/emails`

Стили для почтовых шаблонов (баннеры, кнопки, карточки, текстовые блоки и др.).

### 📁 `/<page>`

Стили для конкретных страниц или крупных тематических секций.  
Например:
- `main/` — главная страница,
- `content/` — типовая контентная страница.

При добавлении новой страницы создаётся собственная папка.

--- 

Точка входа — `index.sass`, который импортирует все необходимые части.
