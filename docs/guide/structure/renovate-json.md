# 📄 renovate.json

Файл конфигурации для Renovate Bot, инструмента автоматического обновления зависимостей проекта.

## Содержимое

```json
{
	"$schema": "https://docs.renovatebot.com/renovate-schema.json",
	"extends": [
		"config:recommended"
	]
}
```

## Пояснения

- `$schema` — ссылка на JSON-схему Renovate для валидации конфигурации.
- `extends` — используется стандартный рекомендуемый набор правил `config:recommended`.
  - автоматически включает обновления зависимостей (npm, devDependencies) по безопасной стратегии.
  - создает Pull Request с обновлениями только после проверки совместимости.

## Зачем нужен

- **Автоматизация поддержки зависимостей** — Renovate проверяет новые версии пакетов и предлагает обновления.
- **Безопасность и стабильность** — обновления проходят через CI и код-ревью, минимизируя риск поломки проекта.
- **Простота конфигурации** — достаточно использовать рекомендованные правила, чтобы бот работал сразу «из коробки».
