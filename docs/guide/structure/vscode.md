# 📁 .vscode

Папка `.vscode/` хранит настройки редактора [Visual Studio Code](https://code.visualstudio.com/) для проекта.  
Она помогает команде использовать единые плагины, правила и сниппеты.

Основные файлы:

- `settings.json` — локальные настройки редактора (например, форматирование, интеграция с линтерами, параметры
  отступов).
- `extensions.json` — список рекомендуемых расширений, которые VS Code предложит установить участникам проекта.
- `global.code-snippets` — общие сниппеты, доступные во всех языках.
- `snippets/*.json` — кастомные сниппеты под конкретные технологии проекта (например, `vue.json`, `javascript.json`,
  `jade.json`).

### Важный момент

- **Общие настройки проекта** (`settings.json`, `extensions.json`, сниппеты) стоит хранить в репозитории — это повышает
  единообразие разработки.
- **Личные настройки каждого разработчика** лучше не класть в git (обычно это решается настройкой `.gitignore`).

Таким образом, `.vscode/` упрощает onboarding и делает работу команды более согласованной.
