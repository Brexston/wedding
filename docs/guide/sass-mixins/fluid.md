# fluid
Это резиновое (гибкое) значение, то есть автоматическое изменение размера шрифта в зависимости от ширины экрана. Цель — сделать так, чтобы текст выглядел гармонично на любом устройстве: от маленького телефона до большого монитора.

## Что такое fluid?

Обычный фиксированный `font-size` вроде `20px` выглядит либо слишком крупно на телефоне, либо слишком мелко на большом экране.  
Fluid решает это с помощью гибких значений, которые плавно масштабируются вместе с шириной вьюпорта.

Современный способ — использовать функцию `clamp()`:

```sass
font-size: clamp(1rem, 2vw + 1rem, 3rem)
```

Это означает:

- Минимум: `1rem`
- Вьюпорт: `2vw + 1rem` (зависит от ширины экрана)
- Максимум: `3rem`

📐 Иными словами:
> Пока экран маленький — шрифт 1rem.  
> С изменением экрана — шрифт становится больше/меньше.  
> Но не превышает 3rem, даже на огромном экране.

| Ширина экрана | Размер шрифта   |
|---------------|-----------------|
| 320px         | 1rem            |
| 768px         | ~1.6rem         |
| 1440px        | ~2.5rem         |
| 1920px+       | 3rem (макс)     |

Такой подход избавляет от ручных `media queries` и делает адаптивное значение чище и удобнее.

Но пользоваться нативным clamp() не очень удобно, так как надо постоянно вычислять вьюпорт, поэтому для этого есть миксин fluid, который высчитывает все значения и выдает итоговую функцию clamp()

::: tip Пример использования в стилях:
`font-size: fluid(24, 16, 320, 1920, 'vw')`
:::

## Аргументы

| Название            | Тип      | По умолчанию | Описание                                       |
|:--------------------|:---------|--------------|:-----------------------------------------------|
| maxSize<sup>*</sup> | `number` | -            | максимальное значение в пикселях               |
| minSize<sup>*</sup> | `number` | -            | минимальное значение в пикселях                |
| minBreakpoint       | `number` | 375          | минимальная ширина точки остановки в пикселях  |
| maxBreakpoint       | `number` | 1920         | максимальная ширина точки остановки в пикселях |
| unit                | `string` | 'vw'         | единица измерения для расчета значения         |
